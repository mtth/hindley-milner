setEntry :: Entry -> Grid s -> ST s (Maybe Conflict) Source #

Attempts to add a new entry to the grid. Note that this method does *not* check that a word is valid. If the word is too long, this method will error.

unsetLastEntry :: Grid s -> ST s (Maybe Entry) Source #

Removes the last added entry, or does nothing if the grid doesn't contain any entries.

candidates :: Grid s -> ST s [Candidate] Source #

Returns all candidate locations for adding new words to the grid. Note that word expansions are not considered valid candidates (e.g. T transforming CAT into CATS would not be returned).
