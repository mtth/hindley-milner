Description

This module implements expression typing. It is the second step in the interpretation process (after parsing in Parse and before evaluation in Eval).

Type inference is done in two phases:

  • First, we walk the AST, assign fresh type variables where appropriate, and emit constraints.
  • Then, we unify all types using the emitted constraints and output the resulting expression's type.

Note that let bindings are generalized:

let f x = x in if (f True) (f 1) 0 -- OK.
let g f = if (f True) (f 1) 0 in let f x = x in g f -- Not OK ("generalized" once in @g f@).
