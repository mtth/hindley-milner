Unsafely

unsafeEnv :: Iden -> (Type, Value) -> Env Source #

Generates an environment with a single identifier bound. The type and value must match, no checks are performed.

This is useful in combination to add "built-in" functionality. For example, to generate an environment with a sin operation, we could do something similar to:

env = unsafeEnv "sin" (tp, ClosureV fn) where
  tp = doubleType -> doubleType
  fn (LitV (DoubleL d)) = LitV $ DoubleL $ sin d
  fn _ = UndefinedV
